# GLB models

Thư mục này chứa các file `.glb` mẫu để demo chạy được ngay sau khi cài dependency.

Bạn có thể thay bằng model thật của bạn:

1. Copy file GLB vào thư mục này, ví dụ: `nha-truyen-thong.glb`.
2. Mở `data/places.js`.
3. Đổi `modelUrl` thành `/models/nha-truyen-thong.glb`.

Lưu ý: file trong thư mục `public` của Next.js sẽ được serve trực tiếp từ root URL.
Ví dụ `public/models/demo-building.glb` sẽ có URL là `/models/demo-building.glb`.
