# GLB models

Chứa các file `.glb` được dùng bởi các địa điểm trong `data/places`.

Khi thêm model mới:

1. Đặt file trong `public/models`.
2. Khai báo đường dẫn tương ứng ở trường `model` của file Markdown.

Ví dụ: `public/models/29-Ha-Noi/Chua_Mot_Cot.glb` dùng `model: 29-Ha-Noi/Chua_Mot_Cot.glb`.
